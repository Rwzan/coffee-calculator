# ☕ Coffee Needed Calculator

A fun mini website that calculates how many cups of coffee you need based on study hours.

Live Link: https://rwzan.github.io/coffee-calculator/

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 💡 How it works
The user enters number of hours, and the app estimates coffee needs using a simple formula.

## 📸 Example
If you work 6 hours → you need 3 cups of coffee ☕
